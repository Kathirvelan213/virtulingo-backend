# virtulingo-backend
backend for the virtulingo-unity applicaition

## Running Locally
To run this project locally, 
- Install uv in your system. [Instructions here](https://docs.astral.sh/uv/getting-started/installation/)
- Clone the repository
```bash
git clone https://github.com/Kathirvelan213/virtulingo-backend
```
- Inside the repository, set up venv
```
uv venv
```
- Activate the virtual environment
```bash
source .venv/bin/activate
```
or
```ps1
& .\.venv\Scripts\activate
```
- Install dependencies
```
uv sync
```
- Start project

